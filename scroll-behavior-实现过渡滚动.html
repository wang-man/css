以往我们要给页面增加一个平滑滚动效果，例如回到顶部，要么用a标签生硬的跳转回顶部，要么只有借助js写上很多行来实现。其实现在大可不必那么麻烦，css就自带这样的天生用来让滚动变得丝滑的属性：scroll-behavior[bɪ'heɪvjə(r)]

语法:


      scroll-behavior: auto;     默认值无效果
      scroll-behavior: smooth;   平滑效果

      就这么简单，强大而实用

像这样的体验增强属性，不管用不用的上，浏览器兼容性如何，其实直接在代码里加上即可，不会有什么影响，浏览器不兼容也没有影响，举个例子
在PC浏览器中，网页默认滚动是在html标签上的，移动端大多数在 body标签上，于是，我加上这么一句：
html, body { scroll-behavior:smooth; }
