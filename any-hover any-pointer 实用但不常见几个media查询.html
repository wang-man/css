之前我们判断当前设备

语法:


      any-hover: 
      none  当前设备不支持鼠标悬停。
      hover 当前设备支持鼠标悬停。

      例如：
      @media (any-hover: none) {
        a {
          text-decoration: none;
        }
      }

any-hover媒体查询可以用来用来精确控制不同设备上的hover交互行为，因此对于跨平台的网页，响应式网站，非常有用。
